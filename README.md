## Desafio "Vive el verano"

![imagen del desafio](assets/img/screenshot.png)

<hr>

### Elementos interactivos:
- Hover:
  - sol,
  - titulo,
  - párrafo,
  - quitasol,
  - hojas palmera,
  - uno de los cocos de cada palmera,
  - mujer,
- Clic:
  - quitasol,
  - el coco "activo" de cada palmera,
  - mujer.
